## Overview {#overview}

inoLED micro is built around the **ESP32-C3** microcontroller and optimized for:

- Addressable LED strips and matrices
- Interactive LED installations
- Home automation lighting
- Architectural lighting
- IoT-connected lighting systems

It provides:

- High-current LED output (10A fused)
- Wide input voltage range (5–24V DC)
- ESD protection on all I/O
- Integrated stabilization capacitor (1000µF)
- USB-C programming interface
- Dedicated boot and reset buttons

The board supports both:

- Single-wire timing protocols (WS2812 type)
- SPI-based LEDs (APA102, WS2801, etc.)

## What Are Intelligent (Addressable) LEDs?

Addressable LEDs allow individual control of each LED’s:

- Color
- Brightness
- Animation behavior

Each LED contains an integrated microcontroller that receives digital data and forwards the remaining signal to the next LED in the chain.

### Key Advantages

- 24-bit color depth (16+ million colors)
- Individual pixel control
- Daisy-chain wiring
- High brightness and rich color rendering
- Stable illumination with internal memory

## Power & Consumption {#power}

### Power Supply

- LED logic: 5V DC
- Supported strip voltages: 5V / 12V / 24V
- Recommended multi-point injection for long strips

### Consumption

- ~60 mA per LED at full white
- 144 LED/m strip can exceed 8A at 5V
- Large installations may require dedicated PSU

### Integrated Protection

- 10A fuse
- 1000µF electrolytic capacitor
- ESD protection on I/O lines

For high-current installations:

- Use thick power wiring
- Inject power at multiple points
- Avoid long thin supply traces

## Install WLED {#wled}

inoLED micro supports direct online flashing of WLED firmware.

### Online Installation

Use:
https://install.wled.me/

Supported browsers:
- Microsoft Edge
- Google Chrome (Chromium-based)

### Installation Procedure

1. Disconnect external power supply.
2. Connect device via USB-C.
3. Open https://install.wled.me/.
4. Select WLED version.
5. Click **Install**.
6. Choose COM port labeled:
   `USB JTAG/serial debug unit (COMxx)`
7. Click **Connect**.
8. Click **INSTALL WLED**.
9. Confirm installation.
10. Wait until flashing completes.
11. Disconnect USB.
12. Connect to external power.

Done.

---

## COM Port Not Found {#bootmode}

If the COM port does not appear:

1. Press **RESET** and **FLASH** simultaneously for 3 seconds.
2. Release RESET.
3. After 1 second release FLASH.
4. Device enters bootloader mode.
5. COM port will appear.
